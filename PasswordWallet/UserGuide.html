🚀 Getting Started

First Launch

When you first open PasswordWallet, you'll be guided through the initial setup:

1. Setup Master Password

  • Create a strong master password (this will be used to unlock the app)
  • Important: Remember this password! There's no recovery method if you forget it
⚠️ Password Requirements:
  • At least 8 characters long
  • Contain a mix of letters, numbers, and symbols
  • Not easily guessable

2. Enable Biometric Authentication (Optional)

  • After setting your master password, you can enable fingerprint or face recognition
  • This allows faster access while maintaining security
  • You can enable/disable this later in Settings

3. Initial Login

  • Use your master password (or biometric if enabled) to access the app
✅ Pro Tip: Write down your master password and store it in a secure physical location (safe, locked drawer) as a backup.

Category Editor Screen

Access: Home → Category Editor

✏️ Add New Category

  1. Tap the "+ Add New Category" button (green, bottom-right)
  2. Enter category name
  3. Choose an icon from the icon picker
  4. Select a color
  5. Tap Save

✏️ Edit Category

  1. Tap on any existing category
  2. Modify name, icon, or color
  3. Tap Save Changes

🗑️ Delete Category

  1. Tap on the category you want to delete
  2. Tap the Delete button (red, trash icon)
  3. Confirm deletion
⚠️ Warning: Deleting a category will delete all password items within it. This action cannot be undone.

⚙️ Manage Category Fields

  1. Tap on a category
  2. Tap "Manage Fields" button
  3. Opens the Category Field Editor screen
  4. Here you can:
    • Add new fields (e.g., "Security Question", "Account Number")
    • Edit existing field names
    • Delete fields
    • Reorder fields (drag and drop)
    • Toggle field masking (hide sensitive fields like passwords)

🔍 Search Functionality

PasswordWallet provides powerful search across all screens where items are displayed.

Where Search is Available:

  • Category Password List Screen
  • Favorites Screen

How to Use Search:

  1. Tap the search icon (magnifying glass) in the app bar
  2. The title bar transforms into a search field
  3. Type your search query
  4. Results filter in real-time as you type

What Search Looks For:

  • Item names (e.g., "Gmail")
  • Field values (usernames, emails, notes, etc.)
ℹ️ Note: Search is case-insensitive. "GMAIL", "gmail", and "Gmail" all match the same items.

Exit Search:

  • Tap the X icon (close) in the app bar
  • Or tap the back button
  • Search clears and full list is restored

🛡️ Security Features

PasswordWallet takes security seriously. Here's how your data is protected:

Local Encryption

  • All password data is encrypted on your device
  • SQLite database with encrypted storage
  • Field values stored securely

Master Password

  • Required to access the app
  • Used as encryption key derivation base
  • Never stored in plain text
  • Hashed using secure algorithms

Biometric Authentication

  • Optional fingerprint/face recognition
  • Faster than typing password
  • Biometric data never leaves your device
  • Fallback to master password always available

Field Masking

  • Sensitive fields (passwords) are masked by default
  • Manual toggle to reveal temporarily
  • Prevents shoulder surfing
  • Copy to clipboard without revealing on screen
🔒 Privacy: PasswordWallet does not collect analytics or telemetry. All data stays on your device (unless you enable cloud sync).

🔧 Troubleshooting

Problem: Forgot Master Password

Solution:
  • No built-in recovery method (for security)
  • If you have a backup file:
    1. Uninstall and reinstall the app
    2. Set up a new master password
    3. Restore from backup file
  • Prevention: Write down your master password and store securely

Problem: Biometric Not Working

  • Check device biometric settings (Settings → Security → Fingerprint/Face ID)
  • Re-enroll biometric if needed
  • Try disabling and re-enabling in PasswordWallet Settings
  • Fallback: Use master password to login

Problem: Backup File Not Restoring

  • Ensure backup file is not corrupted
  • Check file extension (should be .json)
  • Try re-downloading backup from cloud storage
  • Verify backup was created successfully (file size > 0 bytes)

Problem: CSV Import Count Mismatch

Issue: Exported 240 rows, but only 238 items imported

Explanation:
  • Combined CSV uses one row per field
  • Item with 5 fields = 5 rows in CSV
  • Duplicate field rows overwrite previous values
  • Empty item name rows are skipped
Solution: Use split ZIP export for 1:1 item→row mapping

Problem: App Performance Slow

  • Delete old/unused items
  • Export to CSV and reimport to compact database
  • Restart the app
  • Clear app cache (device Settings → Apps → PasswordWallet → Clear Cache)

Problem: Google Drive Sync Not Working

  • Check internet connection
  • Sign out and sign back into Google account
  • Re-authorize app in Google account settings
  • Check Google Drive storage quota

Organization Tips

  • Descriptive Names: "Gmail Personal" vs. "Gmail Work"
  • Leverage Categories: Keep similar items together
  • Use Favorites: Star frequently accessed passwords
  • Add Notes: Use notes field for security questions, backup codes
  • Consistent Naming: Use a naming convention across all items

Backup Strategy (3-2-1 Rule)

  1. 3 copies of your data
  2. 2 different storage types (local + cloud)
  3. 1 offsite backup (cloud or external drive)
✅ Best Practice: Test your restore process occasionally to verify backups work before an emergency.

❓ Frequently Asked Questions

Q: Is my data sent to the cloud?

A: No, unless you explicitly enable Google Drive sync. All data is stored locally by default.

Q: Can I use PasswordWallet on multiple devices?

A: Yes, use the backup/restore feature or Google Drive sync to share data across devices.

Q: What happens if I forget my master password?

A: There's no recovery method. You must have a backup file to restore your data.

Q: How secure is PasswordWallet?

A: Uses industry-standard encryption, local storage, and optional biometric authentication.

Q: Can I export my passwords to another password manager?

A: Yes, use CSV export and import into your new password manager.

Q: Can I customize category icons and colors?

A: Yes, in the Category Editor screen.

Q: How do I add a new field to a category?

A: Category Editor → Select category → Manage Fields → Add Field
