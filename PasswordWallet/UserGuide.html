üõ°Ô∏è Security Features

PasswordWallet takes security seriously. Here's how your data is protected:

Local Encryption

  • All password data is encrypted on your device
  • SQLite database with encrypted storage
  • Field values stored securely

Master Password

  • Required to access the app
  • Used as encryption key derivation base
  • Never stored in plain text
  • Hashed using secure algorithms

Biometric Authentication

  • Optional fingerprint/face recognition
  • Faster than typing password
  • Biometric data never leaves your device
  • Fallback to master password always available

Field Masking

  • Sensitive fields (passwords) are masked by default
  • Manual toggle to reveal temporarily
  • Prevents shoulder surfing
  • Copy to clipboard without revealing on screen
üîí Privacy: PasswordWallet does not collect analytics or telemetry. All data stays on your device (unless you enable cloud sync).

üîß Troubleshooting

Problem: Forgot Master Password

Solution:
  • No built-in recovery method (for security)
  • If you have a backup file:
    1. Uninstall and reinstall the app
    2. Set up a new master password
    3. Restore from backup file
  • Prevention: Write down your master password and store securely

Problem: Biometric Not Working

  • Check device biometric settings (Settings ‚Üí Security ‚Üí Fingerprint/Face ID)
  • Re-enroll biometric if needed
  • Try disabling and re-enabling in PasswordWallet Settings
  • Fallback: Use master password to login

Problem: Backup File Not Restoring

  • Ensure backup file is not corrupted
  • Check file extension (should be .json)
  • Try re-downloading backup from cloud storage
  • Verify backup was created successfully (file size > 0 bytes)

Problem: CSV Import Count Mismatch

Issue: Exported 240 rows, but only 238 items imported

Explanation:
  • Combined CSV uses one row per field
  • Item with 5 fields = 5 rows in CSV
  • Duplicate field rows overwrite previous values
  • Empty item name rows are skipped
Solution: Use split ZIP export for 1:1 item‚Üírow mapping

Problem: App Performance Slow

  • Delete old/unused items
  • Export to CSV and reimport to compact database
  • Restart the app
  • Clear app cache (device Settings ‚Üí Apps ‚Üí PasswordWallet ‚Üí Clear Cache)

Problem: Google Drive Sync Not Working

  • Check internet connection
  • Sign out and sign back into Google account
  • Re-authorize app in Google account settings
  • Check Google Drive storage quota
